# autozcelem

Site for [tolkium.net/autozcelem](https://tolkium.net/autozcelem).

## Deploy to GitHub Pages

1. On GitHub, create a new repository named **`autozcelem`** (same as your subpath).
2. Do **not** initialize with README (you already have content).
3. In this folder, add the remote and push:

```bash
git remote add origin https://github.com/YOUR_USERNAME/autozcelem.git
git branch -M main
git push -u origin main
```

4. In the repo: **Settings → Pages** → Source: **Deploy from a branch** → Branch: **main** → Folder: **/ (root)** → Save.
5. After a minute or two, the site will be at `https://YOUR_USERNAME.github.io/autozcelem`. If your custom domain tolkium.net is set up like hmstrsoc, it will also be at **https://tolkium.net/autozcelem**.

Replace `YOUR_USERNAME` with your GitHub username.
